# 00 Standards

Purpose:
This folder defines the standards, templates, naming conventions and lifecycle for SOPs in the SOP Library. It sets the rules that all other SOPs should follow to ensure consistency and reusability.

Scope:
Includes the canonical SOP template, naming convention, and lifecycle guidance for creating, revising, approving and archiving SOPs.

When these SOPs are used:
- When creating new SOPs for any domain in this library.
- When updating SOPs to align with organizational standards.
- When auditing or reviewing SOP quality and compliance.

Related folders:
- All domain folders (01 Infrastructure..07 Operations) should follow the standards defined here.
- 99 Archive for deprecated SOPs.
