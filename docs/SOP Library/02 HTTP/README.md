# 02 HTTP

Purpose:
This folder contains SOPs for HTTP integration patterns: constructing requests, authentication, response parsing, pagination, rate limits, retries, and data transformation.

Scope:
Applies to HTTP-based integrations and API interactions used by integration scenarios.

When these SOPs are used:
- When implementing HTTP requests to external services.
- When handling common HTTP patterns such as pagination, authentication and rate limiting.

Related folders:
- 01 Infrastructure for webhook and connection setup.
- 03 Make for scenario-level HTTP modules.
- 06 Testing for API and webhook testing procedures.
