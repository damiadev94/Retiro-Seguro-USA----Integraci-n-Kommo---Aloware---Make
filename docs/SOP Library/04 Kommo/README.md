# 04 Kommo

Purpose:
This folder contains SOPs specific to Kommo (formerly amoCRM) integrations: authentication, searching contacts and leads, notes, updates, custom fields, and webhooks.

Scope:
Applies to integrations that interact with Kommo APIs and runtime integrations that manipulate contacts, leads and related objects.

When these SOPs are used:
- When integrating with Kommo for creating, updating or searching CRM records.
- When configuring Kommo-specific authentication and webhook handling.

Related folders:
- 02 HTTP for API patterns.
- 03 Make for scenario implementations that interact with Kommo.
- 06 Testing for Kommo test procedures.
