# 03 Make

Purpose:
This folder contains SOPs covering Make.com (formerly Integromat) scenarios: creating scenarios, routers, filters, iterators, aggregators, scheduling, variables, error routes, logging and debugging.

Scope:
Applies to scenario design, configuration and runtime practices for Make scenarios used in integration projects.

When these SOPs are used:
- When building or modifying Make scenarios for integrations.
- When designing routing, error handling and scheduling behaviors.

Related folders:
- 01 Infrastructure for connection and webhook configuration.
- 02 HTTP for HTTP modules used inside scenarios.
- 06 Testing for scenario testing practices.
