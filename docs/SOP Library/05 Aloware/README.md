# 05 Aloware

Purpose:
This folder contains SOPs specific to Aloware integrations: API authentication, contact creation and update, dialing sequences, agent search, and webhook handling.

Scope:
Applies to integrations that interact with Aloware APIs and runtime flows related to telephony and contact management.

When these SOPs are used:
- When building integrations that create or update contacts, assign dialing sequences, or interact with Aloware agents.
- When configuring Aloware webhooks and authentication.

Related folders:
- 02 HTTP for API patterns.
- 03 Make for scenario workflows that use Aloware.
- 06 Testing for Aloware test recipes.
