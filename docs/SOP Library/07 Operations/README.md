# 07 Operations

Purpose:
This folder contains operations-focused SOPs: deployment, rollback, monitoring, incident response, log analysis and maintenance for integration systems.

Scope:
Applies to runbook-level operational procedures executed by DevOps, SRE, or platform teams supporting integrations.

When these SOPs are used:
- During deployment, incident response, or routine maintenance of integration infrastructure and scenarios.
- For monitoring setup and log analysis procedures.

Related folders:
- 01 Infrastructure for environment-level actions.
- 06 Testing for pre-deployment validation steps.
- 99 Archive for retired operational procedures.
