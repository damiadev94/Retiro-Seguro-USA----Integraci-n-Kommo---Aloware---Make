# 99 Archive

Purpose:
This folder stores deprecated, superseded, or retired SOPs. It preserves history and reference information for procedures no longer actively used.

Scope:
Contains archived SOPs that should not be referenced by new runbooks. Archived items include pointers to the current SOP when available.

When these SOPs are used:
- For audit, historical reference, or rollback context when investigating past changes.
- When a SOP is replaced, the previous version is moved to this folder and marked accordingly.

Related folders:
- 00 Standards for lifecycle rules that govern archiving.
- All domain folders (01..07) where active SOPs are maintained.
