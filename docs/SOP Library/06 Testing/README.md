# 06 Testing

Purpose:
This folder contains SOPs for testing integration components: unit tests, integration tests, webhook tests, API testing, error simulation and validation checklists.

Scope:
Applies to automated and manual testing procedures used to validate integrations before deployment.

When these SOPs are used:
- During development and QA cycles to validate scenario behavior and API interactions.
- When simulating errors and validating recovery and retry logic.

Related folders:
- 02 HTTP for API test patterns.
- 03 Make for scenario-level testing.
- 01 Infrastructure for environment and webhook testing.
