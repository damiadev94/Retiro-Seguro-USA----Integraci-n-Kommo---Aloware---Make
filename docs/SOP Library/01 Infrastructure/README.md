# 01 Infrastructure

Purpose:
This folder contains infrastructure-related SOPs used by integration projects (connections, webhooks, environment configuration, secrets, logging, and environment naming conventions).

Scope:
Procedures apply to configuring integration platform infrastructure and shared runtime components used across projects.

When these SOPs are used:
- During initial environment setup and connection creation.
- When configuring shared platform-level components such as webhooks, secrets, logging, and retry policies.
- When preparing environments for deployment and operations.

Related folders:
- 00 Standards for naming and template guidance.
- 03 Make for scenario-level configuration that depends on platform infrastructure.
- 99 Archive for deprecated infra SOPs.
